# Internship-data-analysis-project
YouTube Channel Data Analysis
Project Overview

This project presents a comprehensive analysis of the world's top YouTube channels based on key performance metrics such as total views, subscriber counts, video uploads, and content categories. The study aims to identify the factors that contribute to channel popularity and audience engagement on YouTube.

By analyzing channel statistics and visualizing important trends, the project provides insights into content performance, audience preferences, and the growth of different content categories across the platform.

Objectives
Analyze top YouTube channels using various performance metrics.
Identify relationships between subscribers, views, and uploaded videos.
Compare the popularity of different content categories.
Examine geographical trends among successful YouTube creators.
Generate meaningful visualizations to support data-driven insights.

Key Findings
Music and Entertainment channels dominate total viewership.
High subscriber counts do not always guarantee higher average views per video.
Regional creators are increasingly competing with global channels.
Consistent content production contributes significantly to audience growth.
